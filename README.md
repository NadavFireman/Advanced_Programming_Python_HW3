# Advanced Programming: Pandas Data Analysis (HW3)

This repository contains the solution for Home Assignment 3 in the Advanced Programming course (Shenkar College, IE&M).

The assignment focuses on mastering the Pandas library for efficient data manipulation, cleaning, and complex querying of the provided CSV dataset.

## Key Skills Demonstrated

* **Pandas DataFrames:** Reading, cleaning, and structuring data from a **CSV file (`flights.csv`)** into a powerful DataFrame object.
* **Data Cleaning & Preprocessing:** Handling missing values, converting data types, and ensuring data quality.
* **Complex Data Querying:** Implementing sophisticated filtering, grouping, and aggregation operations (e.g., calculating median, mean, and frequency distributions).
* **Statistical Insights:** Extracting meaningful business/operational insights by analyzing flight data metrics.

## Repository Content

* **`Advanced_Programming_HW3.ipynb`**: Full Python solution notebook with code and analysis results.
* **`flights.csv`**: The raw dataset used for the analysis.
* **`Assignment_Instructions_3.pdf`**: Original assignment instructions.
